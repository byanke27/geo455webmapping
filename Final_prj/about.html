About the Map

Project background, data sources, and methodology.

Project Overview

This interactive web map visualizes Division I FBS college football programs across the United States, exploring whether increased financial investment in football programs correlates with better on-field performance. Using data spanning the 2022, 2023, and 2024 seasons, users can explore stadium locations, compare program spending, and examine win-loss records across all 128 FBS programs.

The financial landscape of college athletics has shifted dramatically in recent years, particularly following the introduction of Name, Image, and Likeness (NIL) policies allowing student-athletes to receive compensation. This map provides a spatial lens on those trends, letting users identify regional patterns and outliers in the relationship between spending and performance.

How to Use the Map

  • Stadium Markers — Click any marker to see team name, stadium, city, conference, capacity, and year built.
  • Proportional Circles — Circle size represents stadium capacity. Hover to highlight and see a quick popup.
  • Heat Map — Heat intensity is weighted by stadium capacity, showing regional density of large programs.
  • Cluster Markers — Zoomed out view groups nearby stadiums into clusters. Zoom in to expand them.
  • Search Bar — Search by team name (e.g. "Michigan" or "Alabama") to zoom directly to that stadium.
  • Basemap Toggle — Switch between Satellite, Streets, and Light basemaps using the layer control panel.
  • Home Button — Returns the map to the default full US extent.
  • Mini Map — The small map in the bottom left provides geographic context while you explore.

Methodology

Stadium coordinates from Kaggle were used as the spatial foundation for the project. Financial spending totals from the Knight-Newhouse College Athletics Database and win-loss records from Sports Reference were cleaned and standardized using Python, then joined to stadium locations using school name as the common key.

Out of 128 FBS stadiums, 107 were successfully matched with financial data and 115 with win-loss records. Schools without a match — typically due to naming differences between datasets — display available fields and show "N/A" for missing values. All data was compiled into a single JavaScript file for direct browser delivery without requiring a web server.
