About the Map

Project background, data sources, and methodology.

What This Map Is

This interactive web map visualizes all 128 Division I FBS college football programs across the United States, exploring whether increased financial investment in football programs correlates with better on-field performance. Using data from the 2022, 2023, and 2024 seasons, users can compare program spending, win-loss records, and AP Poll rankings at each stadium location across the country. The intended audience includes sports fans, students, and anyone curious about the financial side of college athletics who wants to explore regional patterns and outliers beyond what static rankings and data tables can show.

The financial landscape of college athletics has shifted dramatically following the introduction of Name, Image, and Likeness (NIL) policies, driving programs to invest more to stay competitive in recruiting and performance. Unlike existing sources like the Knight-Newhouse database or Sports Reference, this map integrates spending, performance, and location into a single interactive geographic interface — letting users see at a glance whether the biggest spenders are also the biggest winners.

How to Use the Map

  • Stadium Markers — Click any stadium icon to see team name, stadium, city, conference, capacity, and year built.
  • Proportional Circles (Spending) — Circle size represents total football program spending for the selected year. Teams with a gold ring finished the season ranked in the AP Top 25.
  • Performance Layer (Win %) — Circle size represents program spending while fill color shows win percentage. Green = high win rate, red = low win rate. Use this layer to assess whether spending translates to wins.
  • Year Dropdown — Select 2022, 2023, or 2024 in the right sidebar to update both circle layers for that season.
  • Search Bar — Type a team name (e.g. "Michigan" or "Alabama") in the top right to zoom directly to that stadium.
  • Basemap Toggle — Switch between Light, Satellite, and Streets basemaps using the layer control panel in the top right of the map.
  • Home Button — Click the globe icon to reset the map to the full US extent.
  • Mini Map — The small overview map in the bottom left provides geographic context while exploring zoomed-in views.

Methods

  • Filter: Stadium dataset filtered to FBS programs only, reducing from 253 total stadiums to 128.
  • Clean and standardize: All source files loaded into Python using pandas. A manual school name mapping dictionary resolved naming differences across datasets (e.g. "The Ohio State University" → "Ohio State").
  • Join: Financial spending and win-loss records joined to stadium locations using school name as the common key. 107 of 128 matched spending data; 115 of 128 matched win-loss records. Unmatched schools display N/A in popups.
  • Field selection: Fields retained per feature — team, stadium, city, state, conference, capacity, year built, spending/wins/losses/win percentage/AP ranking for each year (2022–2024).
  • Simplify geometry: Not applicable — point data only. Coordinates rounded to 5 decimal places. CRS: WGS84 (EPSG:4326), standard for Leaflet.js.
  • Web delivery: Exported as an embedded JavaScript variable (stadiums.js, ~63 KB) rather than a GeoJSON fetch() call, allowing the map to load correctly in all environments including local file access.
  • Performance decisions: Only Stadium Markers load by default. Proportional Circles, Performance Layer, and Heat Map are user-toggled to reduce initial visual load. No clustering or geometry simplification was needed at 128 point features.

Limitations & Credits

Limitations
  • 21 of 128 FBS programs are missing spending data due to naming mismatches between datasets. These schools display N/A for spending fields.
  • 13 of 128 programs are missing win-loss records for the same reason.
  • AP Poll rankings reflect end-of-season final rankings only, not preseason or in-season polls.
  • Spending data reflects total football program expenditures and may include facilities, coaching salaries, and administrative costs — not just direct on-field investment.
  • The red-green color encoding for win percentage may be difficult for users with color vision deficiency. A future version would add a secondary visual cue.
